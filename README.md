## Sobre o projeto

Página de apresentação e lore do Safi'jiiva, construída como fan site da série Monster Hunter. O site cobre a história, habilidades e ecologia do monstro com uma identidade visual inspirada em sua aparência — vermelho carmesim e azul safira.

**Seções:**
- Hero com vídeo de fundo em loop
- Registro do Bestiário — lore completa
- Perfil do Monstro — ficha com atributos
- Sapphire of the Emperor — técnica definitiva
- Classificação Ecológica

---

## Tecnologias

- HTML5
- CSS3 (custom properties, grid, animações)
- JavaScript vanilla

Sem frameworks, sem dependências.
